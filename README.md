## ğŸ“– Deskripsi Program

Program ini menunjukkan penggunaan **Exception Handling (try, catch, finally)** dalam sistem rekening sederhana.

### ğŸ§© Fitur Utama

* Melihat saldo rekening
* Melakukan debit (penarikan)
* Melakukan kredit (penambahan saldo)
* Menangani error jika input tidak valid atau saldo tidak cukup

### âš™ï¸ Konsep yang Digunakan

| Konsep                 | Implementasi                                             |
| ---------------------- | -------------------------------------------------------- |
| **Encapsulation**      | Atribut `nama`, `nomor`, dan `saldo` bersifat private    |
| **Exception Handling** | Menangani kesalahan input dan logika dengan `try-catch`  |
| **Finally Block**      | Menampilkan pesan akhir program setelah eksekusi selesai |
